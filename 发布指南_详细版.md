# 🌍 属于你们的旅行日记 - 详细发布指南

这就带你把这个网页发布到互联网上，生成一个专属链接，你可以发给宝宝，也可以发给家人朋友看。

---

## ✅ 第一步：最后检查你的文件
确保你的 `travel-diary` 文件夹里有以下内容，缺一不可：

1.  **核心代码**：
    *   `index.html` (网页入口) 
    *   `styles.css` (样式)
    *   `script.js` (功能逻辑)
    *   `data.js` (你的旅行数据)

2.  **资源文件夹**（必须和上面代码在同一层级）：
    *   📂 `Inter-Data` (里面放国际旅行的照片，如 `2025Spain.jpg`)
    *   📂 `Dome-Data` (里面放国内旅行的照片，如 `2025Shanghai.jpg`)
    *   📂 `music` (里面必须有 `Lost Stars.mp3`)

---

## 🚀 第二步：创建 GitHub 仓库
1.  打开浏览器，访问 [github.com](https://github.com)。
2.  **注册/登录**：如果没有账号，用邮箱注册一个（免费的）。
3.  **新建仓库**：
    *   登录后，点击右上角的 **+** 号，选择 **New repository**。
    *   **Repository name** (仓库名)：起个英文名，比如 `our-love-map` 或 `travel-diary`。
    *   **Public** (公开)：必须选这个，否则免费版不能发布网页。
    *   点击最底下的绿色按钮 **Create repository**。

---

## 📤 第三步：上传文件
1.  **找到上传入口**：
    *   **情况 A（如果你刚才没有选 README）**：你会看到很多代码命令，找中间一行蓝色小字链接：`…or upload an existing file`。
    *   **情况 B（如果你刚才选了 README）**：页面上已经有文件了，请点击右上角的 **Add file** 按钮，选 **Upload files**。

2.  **拖拽上传**：
    *   打开你电脑上的 `travel-diary` 文件夹。
    *   **全选** 里面的所有文件和文件夹（包括 `Inter-Data`, `music` 等文件夹）。
    *   直接拖进网页的灰色区域。

3.  **等待上传**：
    *   网页会显示上传进度条，等待全部变成绿色。

4.  **保存提交**：
    *   拖到底部，在 **Commit changes** 按钮上方写一句备注（比如 "我的第一次上传"），然后点击绿色的 **Commit changes** 按钮。
    *   *注意：如果文件多，这里可能要处理几分钟，不要关闭页面。*

---

## ⚙️ 第四步：生成网址 (最重要的一步)
1.  上传成功后，点击页面上方的 **Settings** (设置) 图标（通常是个齿轮）。
2.  在左侧菜单栏里找 **Pages** (或者叫 GitHub Pages)。
3.  在 **Build and deployment** 下面的 **Branch** 选项里：
    *   把 `None` 改成 **`main`** (或者 `master`)。
    *   右边的文件夹保持 `/ (root)` 不变。
    *   点击 **Save** (保存)。

---

## 🎉 第五步：大功告成！
1.  保存后，刷新这个 Pages 页面。
2.  你会看到顶部出现一个提示（可能需要等待 1-2 分钟）：
    > "Your site is live at..."
3.  那个链接就是你的专属网址！
    *   例如：`https://jacky123.github.io/our-love-map/`

---

## 🔄 未来通过网页更新日记
如果以后你们去了新地方，想更新地图：
1.  **在 GitHub 上直接改**：
    *   在你的仓库页面点击 `data.js`。
    *   点击右上角的 ✏️ (笔形状) 图标。
    *   按照之前的格式，复制粘贴添加新的地点。
    *   点 Commit changes 保存。网页会自动由 GitHub 更新。
2.  **上传新照片**：
    *   进入 `Inter-Data` 或 `Dome-Data` 文件夹，点击右上角 **Add file** -> **Upload files** 把新照片传上去。

祝你们的旅行地图越来越丰富！✈️❤️
